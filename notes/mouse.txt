Notes on configuring Logitech m500 mouse:
  Useful site: http://blog.hanschen.org/2009/10/13/mouse-shortcuts-with-xbindkeys/
  sudo apt-get install xbindkeys xautomation
  xbindkeys --defaults > $HOME/.xbindkeysrc
  Commands:
    Get button numbers: xev
    Execute mouse and keyboard actions as commands: 'xte' with args, see site
    Restart xbindkeys: kill `pgrep -f '^xbindkeys$'` ; xbindkeys
  In home dir: ln -s unihome/rc/.xbindkeysrc .xbindkeysrc
  xbindkeys is probably already configured to start at system start
